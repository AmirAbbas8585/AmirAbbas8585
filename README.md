### 🛡️ Featured project — Jester Anti Cheat

A packet-based anti-cheat for **Paper / Spigot / Purpur / Folia**, built on the Grim
prediction engine with **40+ extra checks** of my own (combat, scaffold, auto-tools)
and a strong focus on **low false-positives** across mixed client versions and proxies.
